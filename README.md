🧪 Causal A/B-Testing Framework
Ein automatisiertes Sicherheitsnetz für datengetriebene Entscheidungen.

🎯 Falsche Daten in A/B-Tests führen zu teuren Fehlentscheidungen. Dieses Tool ist ein automatischer "Bodyguard", der Rohdaten auf versteckte technische Fehler prüft, bevor das Unternehmen auf dieser Basis strategische Entscheidungen trifft.

💡 Business Value (Was das Tool leistet)
Ressourcen schonen (Pre-Analysis): Berechnet vorab exakt, wie lange ein Test laufen muss. Verhindert Geldverbrennung durch zu lange oder sinnlose Tests.

Schutz vor Frontend-Bugs (SRM Kill-Switch): Erkennt sofort, wenn Website-Traffic falsch verteilt wurde, und stoppt die Analyse, um falsche Schlüsse zu verhindern.

Daten-Reinheit (Spillover-Guard): Filtert vollautomatisch User heraus, die durch technische Fehler in mehreren Testgruppen gleichzeitig gelandet sind.

🛠 Tech Stack
Sprache: Python 3.8+

Data Science: Pandas, SciPy, Statsmodels, DoWhy

Qualitätssicherung: Pytest (Test-Driven Development)
